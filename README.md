# boothub

> Bootstrap any AI agent (Claude Code, Cursor, Codex, Aider…) from one URL.

```text
"Hey Claude, fetch boothub.dev/jhammant and bootstrap me."
```

The agent fetches a manifest, shows a diff, asks for approval, then writes
skills, slash commands, hooks, MCP configs, and `CLAUDE.md`/`AGENTS.md` into
your project — and stands up a multi-agent **swarm** if the profile bundles one.

Profiles are just GitHub repos. There is no account, no install, no CLI.
