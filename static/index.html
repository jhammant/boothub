🥾 boothub

Bootstrap any AI agent from one URL.

Status: 🚧 pre-alpha. Scaffolding in progress.

The idea

Tell your AI agent (Claude Code, Cursor, Codex, Aider…) to fetch a URL:

"Hey, fetch https://boothub.dev/jhammant and bootstrap me."

The agent reads a manifest, shows a diff, asks for approval, then writes skills, slash commands, hooks, MCP configs, and CLAUDE.md/AGENTS.md into your project — and stands up a multi-agent swarm if the profile bundles one.

How it works

Profiles are GitHub repos: github.com/USERNAME/boothub-profile. The site is a thin manifest router; never executes anything. The agent is the runtime.
